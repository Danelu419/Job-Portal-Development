# Job Portal Development

## 📌 Project Description
A full-fledged Job Portal application developed using Python and Django framework to connect employers with job seekers. The platform enables job posting, job search, and application management in a structured and efficient way.

---

## 🚀 Key Features

### 👤 User Authentication
- Secure user registration and login system
- Role-based access for Job Seekers and Employers

### 💼 Job Listings
- Employers can create and manage job postings
- Job seekers can browse available jobs and apply directly

### 🔍 Search & Filter
- Advanced search functionality
- Filters based on location, job type, experience level, and keywords

### 🔗 RESTful APIs
- Designed and implemented REST APIs using Django REST Framework
- Supports integration with web and mobile applications

### 📊 Application Management
- Track job applications and statuses
- Manage applicant data efficiently for employers
